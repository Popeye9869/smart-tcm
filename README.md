# 智能中医诊疗系统 (Smart TCM)

一个基于人工智能技术的现代化中医诊疗平台，融合传统中医理论与现代AI技术，提供智能诊断、个性化处方推荐和中医知识库服务。

## 🌟 核心功能

### 🤖 智能诊断
- **AI症状分析**：基于大模型的智能症状分析系统
- **四诊合参**：望、闻、问、切四诊信息数字化采集
- **证型识别**：自动识别中医证型，提供辨证分析
- **病因病机**：智能分析疾病发生发展机制

### 💊 个性化处方
- **智能处方推荐**：根据诊断结果推荐个性化中药处方
- **经典方剂库**：集成历代经典方剂，支持加减化裁
- **用药安全**：药物配伍禁忌检查，确保用药安全
- **剂量优化**：根据患者情况智能调整药物剂量

### 📚 中医知识库
- **中药材百科**：详细的药材性味归经、功效主治信息
- **经典方剂**：历代名方解析与现代应用
- **诊断方法**：中医诊断技巧与临床经验
- **理论知识**：中医基础理论与临床应用

### 📋 病历管理
- **完整病历记录**：支持病历的创建、编辑、查询和管理
- **历史记录分析**：智能分析历史诊疗数据
- **数据统计**：诊断统计、用药分析等数据可视化
- **导出功能**：支持病历数据的导出和打印

### 👤 个人中心
- **健康档案**：个人健康信息管理与追踪
- **诊断记录**：个人历史诊断记录查看
- **收藏管理**：收藏感兴趣的知识内容
- **偏好设置**：个性化界面和功能设置

## 🛠️ 技术架构

### 前端技术栈
- **框架**：Vue 3 + TypeScript
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **国际化**：Vue I18n
- **动画效果**：GSAP + VueUse Motion
- **HTTP客户端**：Axios

### 核心特性
- **响应式设计**：完美适配PC、平板、手机等多种设备
- **现代化UI**：优雅的中医主题界面设计
- **流畅动画**：丰富的交互动画效果
- **深色模式**：支持深色/浅色主题切换
- **多语言支持**：中文、英文双语界面

## 🚀 快速开始

### 环境要求
- Node.js: ^20.19.0 || >=22.12.0
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```
访问 http://localhost:5173 查看应用

### 构建生产版本
```bash
npm run build
```

### 代码质量检查
```bash
# 类型检查
npm run type-check

# 代码格式化
npm run format

# 代码检查
npm run lint
```

## 🔧 核心模块说明

### 智能诊断模块 (`src/views/DiagnosisView.vue`)
- 五步诊断流程：基本信息 → 症状描述 → 四诊信息 → 智能诊断 → 治疗方案
- 集成AI大模型API，提供智能分析
- 支持症状的多维度描述和四诊信息采集

### 病历管理模块 (`src/views/RecordsView.vue`)
- 表格和卡片两种视图模式
- 支持病历的搜索、筛选、导出
- 完整的CRUD操作

### 知识库模块 (`src/views/KnowledgeView.vue`)
- 分类浏览和搜索功能
- 详细的知识内容展示
- 相关内容的智能推荐

### 个人中心模块 (`src/views/ProfileView.vue`)
- 个人信息管理
- 诊断记录查看
- 收藏内容管理
- 系统偏好设置
